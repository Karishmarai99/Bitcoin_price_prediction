+# Bitcoin Price Prediction Model

This project predicts Bitcoin prices using Machine Learning techniques in Python.

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Features
- Fetch Bitcoin historical data
- Data preprocessing and analysis
- Visualization of price trends
- Machine learning model training
- Future Bitcoin price prediction

## Project Workflow
1. Download historical Bitcoin data
2. Clean and preprocess the dataset
3. Visualize price trends using Matplotlib
4. Train a Linear Regression model
5. Predict future Bitcoin prices
